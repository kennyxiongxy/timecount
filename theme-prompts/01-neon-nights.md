# Timecount Theme: Neon Nights

## Prompt

A sleek macOS timer application interface with a cyberpunk aesthetic. The design features vibrant neon pink and purple glowing accents on a dark charcoal background. The main timer display shows a circular progress bar with a bright magenta glow, displaying "5:30" in a futuristic digital font. Multiple timer cards are arranged in a grid below, each with neon pink borders and dark backgrounds. The overall vibe is cyber, sleek, and energetic with visible light glow effects around all UI elements. The design should feel like a high-tech command center. Ultra modern, bold typography, smooth gradients, professional product photo quality. 4K resolution.
