# Timecount Theme: Synthwave

## Prompt

A retro 80s synthwave-inspired timer application for macOS. The design features a vibrant gradient background transitioning from deep purple to orange, evoking sunset vibes. The timer cards have glowing neon orange and pink borders with dark backgrounds. The main timer displays bold retro digits in bright orange, with smooth purple-to-orange gradients creating depth. Grid layout of multiple timers, each with prominent neon glow effects. Classic 80s typography, geometric shapes, and a nostalgic yet modern aesthetic. The overall feeling is warm, energetic, and retro-futuristic. Smooth gradients, neon accents, vaporwave color palette. High quality illustration, vibrant colors, professional design.
